Cómo conectar:
1. Abre script.google.com → Nuevo proyecto
2. Copia el script del campo de abajo y pégalo completo
3. Desplegar → Web App · Ejecutar como: Yo · Acceso: Cualquiera
4. Autoriza los permisos → copia la URL del Web App
5. Pégala arriba y haz clic en Probar

Hojas que se crearán/rellenarán automáticamente:
• Datos Larvicultura - M01…M10 → Calidad + PLG + Población + Despacho + Calidad de Agua (1 fila por TQ)
• Control_Tanque M01…M10 → Parámetros OD/°C (1 fila por TQ × hora)
• Datos Larvicultura - CIO → Igual que M01–M10 (módulo CIO)
• Control_Tanque CIO → Parámetros OD/°C del módulo CIO
• Lab_Algas → Registro diario de cultivo de algas
• Maduración Sala → Salas (upsert por Fecha + Sala)
• Maduración Tanques → Tanques (upsert por Fecha + Sala + Tanque)
• Maduración Lotes → Lotes (upsert por Fecha + Sala + Fila)
• BIOMOL → Diagnóstico molecular (append por muestra)
• Registro_Supervisión → Supervisión técnica AsT (upsert por ID)
• Registro_Desinfección → Desinfección M01–M10 + CIO (upsert por Fecha+Módulo+Tipo+Categoría+Elemento)
• Microbiología / Calidad de Agua / Patología en Fresco → módulo Microbiología (reemplazo por sesión)
Datos como Cantidad Sembrada y nombres de tanques SOLO existen localmente — exporta periódicamente para no perderlos si el navegador limpia el almacenamiento.
